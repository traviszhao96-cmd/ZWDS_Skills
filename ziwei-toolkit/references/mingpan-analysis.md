# 命盘分析

## 用途

这份方法用于把一张紫微斗数原盘，稳定地讲成一套有顺序、能落地、能复盘的命格分析。它适合两类场景：

- 实际论命：用户给出生资料或命盘，要求分析命格、六亲、格局、大限。
- 教学训练：用户要学习一套固定的原盘论命步骤。

核心原则只有四条：

1. 先定盘，后论命。
2. 先看体，后看用。
3. 先看原盘格局，再看时间引动。
4. 先回答“是什么”，再回答“为什么”，最后回答“何时应”。

## 固定输出顺序

最终回答尽量收束成这六段：

1. `定盘`
2. `命格主轴`
3. `六亲`
4. `体用落地`
5. `核心宫位与触发点`
6. `大限总论`

如果是教学型回答，可以把这六段展开成“七步法”；如果是实际论命，就按这六段给结果。

## 第0步：定盘

### 目标

先把盘定稳，避免后面所有结论漂移。

### 要做的事

1. 确认历法、性别、时间、是否需要校时。
2. 确认命宫、身宫、来因宫、生年禄权科忌。
3. 标出命宫主星、自化、平方、六内六外。
4. 把六条本对线先定出来：
   - 命迁线
   - 兄友线
   - 夫官线
   - 子田线
   - 财福线
   - 父疾线

### 这一步要说明什么

- 这张盘是否是“当前工作盘”。
- 有没有时辰未验证的风险。
- 后面哪些判断是基于当前盘，哪些判断是已经过六亲验证的。

## 第1步：用来因宫与生年四化定命格主轴

### 目标

先定这个人“一生最像什么”，再谈具体事件。

### 要做的事

1. 先看来因宫。来因宫回答的是：这个人一生的大事，为什么总会聚焦在这些领域。
2. 再看生年四化单点：
   - 禄：资源、得到、顺手、愉悦、给与
   - 权：掌控、执行、专业、推动、责任
   - 科：体面、解释权、名义、修饰、贵人
   - 忌：执着、代价、纠缠、压力、结果
3. 再做“两点一线”：
   - 不只看四化落在哪一宫。
   - 还要看“来因宫飞什么到哪里”。
   - 这样才能把“现象”讲成“原因”。
4. 加三层修正：
   - 命宫主星：看人格风格和表面气质。
   - 双象：看两股能量如何组成互动模式。
   - 独象：四化中哪一个是少数派，它常常是整盘关键点。
5. 看六内六外与真禄假禄：
   - 落我宫，常是能为我所用。
   - 落他宫，常是力量向外耗散或因他人而起。
   - 禄的吉凶不只看禄本身，还要看忌落在哪里。
6. 如果有平方，提前记下来，因为它常常不是枝节，而是主轴。

### 这一步最后一定要收成三句话

1. 这个人的核心性格驱动是什么。
2. 这个人的最大优势在哪里。
3. 这个人的命理核心课题是什么。

## 第2步：有六亲信息时，先定位，再解关系

### 目标

六亲分析不是先讲关系，而是先找到人。

### 要做的事

1. 如果用户给了六亲生肖，先定位六亲用神宫位。
2. 定位顺序固定：
   - 先看生年四化本宫与对宫是否直接命中。
   - 不中，再走有体路用。
   - 再不中，再走禄随忌走或科随权走。
   - 仍不中，再考虑时辰有误。
3. 定位完成后，再讲关系。

### 关系分析只讲三层

1. 宫位空间：这段关系主要落在家庭、事业、外缘、精神、合作，还是资源分配。
2. 星曜特质：对方像什么样的人，彼此用什么方式互动。
3. 四化动力：为什么会亲近、牵制、亏欠、依赖或拉扯。

### 如果没有六亲信息

直接省略这一步，不要为了完整而硬讲。

## 第3步：由体入用，把命格翻成现实问题

### 目标

把“这个人天生是什么样”翻成“这个人会在哪些现实问题上发用”。

### 核心定义

- `体`：生年四化。是先天格局、本质、现象。
- `用`：自化与飞宫。是后天运用、动作、发用方式。

### 要做的事

1. 从生年四化所在宫位起飞，同类追踪：
   - 禄转禄
   - 权转权
   - 科转科
   - 忌转忌
2. 优先看 `忌转忌`，因为它最容易把问题讲具体。
3. 把飞到的宫位，当成对前面“体”的补充说明：原来的问题到底落在家庭、钱、工作、身体还是感情，原来的优势到底拿去帮谁、换什么、伤哪里。
4. 再做第二层承接：
   - 禄随忌走
   - 科随权走
5. 需要时加入 `有用归体`：
   - 事件已经从“用”飞出后，再回头看这颗化星的生年源头宫位。
   - 用来交代事件的来源、关联方、资源出处。

### 河图洛书与合十的使用位置

- `河图 / 合十`：偏静态，拿来看先天格局，哪条线是根本盘。
- `洛书`：偏动态，拿来看问题会怎么动、怎么串、怎么往后发展。

## 第4步：由用归体，结合自化、平方、平衡找核心宫位

### 目标

从一堆问题里，找出真正的主战场。

### 自化怎么用

1. 自化不是单独吉凶，它是变化和引动。
2. 自化一定要回头连到生年四化，不能脱体空讲。
3. 区分两种能量：
   - 离心：向外发散、发用、做出来、放出去
   - 向心：向内牵引、拉回来、凝聚到本宫

### 平方怎么用

平方是高强度结构，不是普通细节。优先级很高。

判断时抓三点：

1. 平方落在哪一宫或哪条线。
2. 它让本宫与对宫哪里“不稳定”。
3. 它后面又引动了谁，特别是生年忌和后续忌转忌。

### 平衡怎么用

当一个宫位的能量不对等时，不要停在原宫，要继续看为了补平衡又飞出什么。

这类飞出的能量常常是“以后释前”：后面飞到哪里，就说明前面那个问题会在哪个场景做实。

### 如何找真正的核心宫位

通常真正的核心宫位，会同时满足几项中的多项：

1. 被来因宫点到。
2. 有生年忌、平方或强自化。
3. 又被飞忌、同位宫、对宫反复串联。
4. 六亲用神也落在这里或冲到这里。
5. 这宫位一动，多个专题一起被牵动。

## 第5步：把原盘问题放进时间轴，先看大限，再谈应期

### 目标

原盘负责定格局，大限负责看哪十年最容易把问题做实。

### 基本顺序

1. 本命定格局。
2. 大限看趋势。
3. 流年找引动。
4. 流月做确认。

### 原盘分析里，大限重点看什么

1. 哪一步大限走到来因宫。
2. 哪一步大限走到生年忌宫。
3. 哪一步大限走到平方宫或其对宫。
4. 哪一步大限走到专题正宫。
5. 哪一步大限走到六亲用神宫位。

### 叠宫法的使用

真正要断应期时，重点不是把所有流年宫都讲一遍，而是看：

- 流年命宫叠在本命哪一宫。
- 同时又叠在大限哪一宫。

这两个宫位交叉出来的主题，就是那一年最容易发生的大事。

### 解释顺序不能乱

走到某宫以后，固定按这顺序讲：

1. 生年四化
2. 自化
3. 飞宫

## 第6步：最后收束成可交付的命格结论

### 目标

避免讲成很多碎象，最后却没有一句话结论。

### 固定收束法

一张原盘讲完，尽量收成五个结论：

1. 一句话命格总论。
2. 三个核心优势。
3. 三个核心风险。
4. 六亲关系主线。
5. 哪几个十年最关键。

## 轴线速查

### 六条本对线

- 命迁线：自我、外界、移动、环境
- 兄友线：亲近同辈、大众人际、合作关系
- 夫官线：婚姻关系、社会角色、事业舞台
- 子田线：家庭、成家、房产、子女、桃花、投资
- 财福线：赚钱、享受、欲望、福报、精神满足
- 父疾线：原生家庭、长辈、学习、证照、身体

### 高频专题线

- 命迁子田：常用于看意外、外出、变动、家内家外的连锁问题
- 夫官线：常用于看婚姻与社会角色如何互相牵动
- 父疾线：常用于看原生家庭、身体、学习与责任感的根底

## 常见误区

1. 还没定盘，就开始下结论。
2. 六亲未定位，就先讲关系好坏。
3. 只讲命宫星曜，不讲来因宫与生年四化。
4. 只讲四化落宫，不讲对宫与轴线。
5. 看到自化就直接断吉凶，不回头找生年四化。
6. 看到流年就直接下结论，不先讲原盘和大限。
7. 说了一堆象意，却没有收成一句话命格总论。
