# 六亲关系分析

## 何时读取本文件

在这些情况下读取：

- 用户要分析和父母、配偶、子女、兄弟姐妹的互动模式
- 用户已经给了六亲生肖，希望先定位再解读关系
- 用户没有要求校时，主要目标是理解关系结构

## 默认工作流

关系分析是 toolkit 的默认主线。

1. 先确定工作盘
   - 若用户已确认时辰，就用当前盘
   - 若用户只给了一个时辰但没有要求校时，也先用当前盘
   - 若用户明确要求校时，先做生肖定位，再决定是否换盘
2. 若用户提供六亲生肖，先读 [kinship-zodiac.md](kinship-zodiac.md) 做定位
3. 若没有生肖，就直接从对应宫位切入，但要说明这是未经过生肖验证的分析
4. 输出时把`定位结论`和`关系分析`分开

## 分析框架

关系分析至少要覆盖三个层次。

### 1. 宫位空间意涵

说明该六亲被定位到哪个宫位，以及这个宫位意味着什么。

常见例子：

- 田宅宫：家庭、陪伴、安顿、内在归属
- 迁移宫：距离、外缘、外部互动、分离感
- 官禄宫：责任、要求、角色压力、秩序
- 福德宫：精神交流、情绪背景、内在牵引
- 夫妻宫：亲密关系、合作、伴侣镜像

### 2. 星曜特质

说明宫内主星、辅星、关键结构如何塑造互动模式。

优先看：

- 主星
- 生年四化所在星
- 关键辅星
- 明显的煞曜或空曜

### 3. 四化心理动力

把四化翻译成关系中的心理驱动力。

- 禄：圆融、资源、给与、情感流动
- 权：推动、主导、要求、掌控
- 科：修饰、理性、名义、体面、解释权
- 忌：执念、约束、纠结、亏欠感、难放手

## 输出格式

建议按这个顺序说：

1. 当前工作盘和前提
2. 六亲定位是否已验证
3. 宫位层面的关系场景
4. 星曜层面的性格与互动模式
5. 四化带来的深层动力
6. 一句总结：亲近、牵制、依赖、拉扯、责任、距离，哪类感觉更强

## 风险提示

- 若时辰没有校验，明确说明分析基于当前盘
- 若生肖定位链条不完整，不要把某个宫位说成唯一正确答案
- 若用户只想看关系，不要擅自扩展成全盘断事
